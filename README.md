## 🛠️ Local Development

### Requirements

- Node.js 22+
- npm 11+
- Git

### Clone the repository

```bash
git clone https://github.com/gcapuano1979/giuseppecapuano.dev.git
```

### Enter project directory

```bash
cd giuseppecapuano.dev
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:3000
```
